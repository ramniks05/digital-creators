# Team Images Folder

Place your team member images in this folder.

## File Naming Convention:
- Director image: `director.jpg` (or `.png`, `.webp`)
- Other team members: `member1.jpg`, `member2.jpg`, etc.

## Supported Formats:
- JPG/JPEG
- PNG
- WebP

## Image Requirements:
- Recommended size: 400x500px or larger
- Aspect ratio: 4:5 (portrait)
- File size: Keep under 500KB for optimal loading

## Current Setup:
- Director image path: `/images/team/director.jpg`
- Update the image path in `src/pages/Team.jsx` if you use a different filename

## How to Add Images:
1. Place your director image in this folder as `director.jpg`
2. The image will automatically appear on the Team page
3. To add more team members, update the `teamMembers` array in `src/pages/Team.jsx`
